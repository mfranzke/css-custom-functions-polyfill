Multiple Conditions with CSS Custom Functions

1. Multi-Breakpoint Responsive Background

@function --multi-responsive-bg() {
  result: lightcoral;
  @media (min-width: 480px) { result: lightblue; }
  @media (min-width: 768px) { result: lightgreen; }
  @media (min-width: 1024px) { result: lightyellow; }
}
This background changes color at multiple breakpoints: 480px, 768px, and 1024px!

2. Multi-Feature Detection Typography

@function --multi-feature-font() {
  result: Arial, sans-serif;
  @supports (font-feature-settings: normal) {
    result: "SF Pro Display", system-ui, sans-serif;
  }
  @supports (font-variation-settings: "wght" 400) {
    result: "Inter Variable", system-ui, sans-serif;
  }
}
This text uses progressively enhanced fonts based on browser capabilities!

3. Accessibility-Aware Sizing

@function --accessibility-aware-size() {
  result: 16px;
  @media (prefers-reduced-motion: reduce) { result: 18px; }
  @media (prefers-contrast: high) { result: 20px; }
}
This text size adapts to accessibility preferences!

4. Multi-Breakpoint Grid Layout

@function --responsive-columns() {
  result: 1fr;
  @media (min-width: 480px) { result: repeat(2, 1fr); }
  @media (min-width: 768px) { result: repeat(3, 1fr); }
  @media (min-width: 1024px) { result: repeat(4, 1fr); }
}

5. Multi-Condition Border & Spacing

@function --adaptive-border-width() {
  result: 1px;
  @media (min-width: 768px) { result: 2px; }
  @supports (border-width: max(1px, 0.1vw)) {
    result: max(1px, 0.2vw);
  }
}
This element has adaptive borders and spacing based on screen size and browser features!

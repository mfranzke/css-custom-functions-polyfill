Enhanced CSS Custom Functions Examples

1. Enhanced Border Properties

@function --enhanced-border-width() {
  result: 1px;
  @media (width >= 768px) { result: 2px; }
  @media (width >= 1200px) { result: 4px; }
}
This element has a responsive border that gets thicker on larger screens!

2. Enhanced Spacing System

@function --responsive-spacing() {
  result: 10px;
  @media (width >= 768px) { result: 20px; }
  @media (width >= 1024px) { result: 40px; }
}
This element's margin and padding scale with screen size!

3. Enhanced Typography

@function --enhanced-font-weight() {
  result: bold;
  @supports (font-variation-settings: "wght" 400) {
    result: 600;
  }
}
This text uses variable fonts when supported, with responsive sizing!

4. Enhanced Background System

@function --enhanced-bg-color() {
  result: #ffffff;
  @media (prefers-color-scheme: dark) {
    result: #2d3748;
  }
}
This background adapts to your color scheme preference and supports modern gradients!

5. Enhanced Visual Effects

@function --enhanced-transform() {
  result: none;
  @supports (transform: translateY(-5px)) {
    result: translateY(-5px);
  }
}
Hover me! This element uses progressive enhancement for transforms and shadows.

6. Enhanced Layout System

@function --enhanced-display() {
  result: block;
  @supports (display: grid) { result: grid; }
}
